# AES-128-brute-force-test-using-python
This project was implemented using Python 3 and the PyCryptodome library, a self-contained cryptographic package. The simulation utilizes the Advanced Encryption Standard (AES) with a 128-bit key in Electronic Codebook (ECB) mode 
Use the code on Python 3 latest version and ensure Pycryptodome is installed on your device
libraries imported are Crypto.* and python's Time.
